Description

GnomenuClientHelper provides fundanmental messaging mechanism for a menu client. As the name indicates, it helps a GlobalMenu client, which eventually is a GtkGlobalMenuBar widget. Usually you never use it. Use GtkGlobalMenuBar instead if you want to take the benefits of a global menu.
